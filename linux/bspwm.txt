-----Atajos------
super => Windows
return => Enter

Super + Return => Abir una nueva terminal
Super + d => Abrir el rofi
Super + alt + q => Resetear el bspwm
Super + w => Cerrar ventana
Super + m => Ventana completa (full screen)
Super + g => Itercambiar el nodo actual por la ventana más grande
Super + t => Regresar la pantalla a su posición
Super + s => Establecer el estado de la pantalla
Super + shift + (Left,Down+Up+Right) => Mover de posición la ventana
ctrl + Super (1-9) => Preseccionado con números
Super + ctrl + alt + (Left,Down+Up+Right) => Preseleccionado
Super + ctrl + alt + space => Cacelar el Preseleccionado
alt + window + (Left,Down+Up+Right) => Resize

return + alt + r => Resetear la consola
Cambiar de ventana => window + 1 +2 +3

ctrl + windows + (Rigth+left+Up)

super + f => FUll screen

setxkbmap -layout latam  => Cambiar el idioma a español latinoamericano
